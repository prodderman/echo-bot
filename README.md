# echo-bot

> Before start, you need to create configuration files for each bot using a template and fill them.  
> Example for Telegram:  
> `config.template.json -> config.json`  
> and then:
> ```json
> {
>   token: "Your token"
> }
> ```
>

## Available Scripts

### `stack build`
Builds the project

### `stack exec bot vk`
Runs bot for VK

### `stack exec bot telegram`
Runs bot for Telegram

### `stack test`
Runs tests
